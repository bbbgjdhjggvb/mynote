# 设置键
```
# 设置 键 值
SET name fufu
# 当name键不存在时才设置
SETNX name fufu

# 获取键
GET name

# 判断键是否存在，存在返回1，不存在返回0
EXISTS name

# 查看所有的键 
KEYS *
```

# 设置带过期时间的键值对
```
# 查看键的过期时间，如果是-1表示没有设置过期时间，-2表示已经过期
TTL name

# 设置键过期时间，10秒后过期
EXPIRE name 10

# 
SETEX name 10 fufu
```

# 删除键
```
# 删除所有的键
FLUSHALL

# 删除单个键
DEL key
```

# 查看key类型
```
type key
```
