### 确认框
```js
confirm("确认内容");
```
1. 如果用户点击确认放回true,点击取消返回false
2. 在用户点击确认或者取消操作之前，会阻止用户对浏览器的所有操作
![[Pasted image 20241107103753.png]]
### 提示框
```js
var num = prompt("输入一个数字", 0);
```
1. 点击取消，返回null
2. 点击确认放回文本框中输入的值
3. 阻塞io，执行这条语句之前不会执行后面语句
