#### 设置背景为渐变色
```css
div{
	background: linear-gradient(to bottom, #xxxx, #xxxx)	
}
```
1. linear-gradient第一个参数
	1. to bottom
	2. to right
	3. to bottom right
2. linear-gradient第二个参数：起始颜色
3. linear-gradient第三个参数：结束颜色
4. 实现效果从起始颜色渐变为结束颜色
