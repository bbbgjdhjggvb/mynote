# 优先队列(堆)
最大堆和最小堆、自动维护队列的有序性

# 基本用法和默认行为
- 头文件`<queue>`
- 默认最大堆
- `pq.push(3)`入队
- `pq.pop()`出队
- `pq.top()`查看队头
```c++
#include <queue>
#include <iostream>

int main() {
    // 默认：最大堆（大元素先出队）
    std::priority_queue<int> pq;
    
    pq.push(3);
    pq.push(1);
    pq.push(2);
    
    while (!pq.empty()) {
        std::cout << pq.top() << " ";  // 输出：3 2 1
        pq.pop();
    }
    return 0;
}
```
