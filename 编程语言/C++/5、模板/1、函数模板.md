#### 格式
```
template<typename T,typename U>
void foo(T arg1,U arg2){
}
```

#### 实例化模板
调用一个函数模板时，编译器用函数实参为我们推断模板的参数的类型，创建出一个新的函数实例
```
foo(1,1.0);
```

#### 显示实例化模板
```
foo<type>(实参)；
```

#### 模板类型参数
1. 逗号分隔
2. 每个类型名都要有typename说明
3. 可用来做返回类型、内部变量类型
#### 模板编译
1. 编译器遇到模板并不生成代码
2. 只有实例化模板才生成代码
3. 为了实例化一个模板、编译器必须知晓全部，所以函数模板的声明和实现必须写在同一个文件，最好写在头文件中
