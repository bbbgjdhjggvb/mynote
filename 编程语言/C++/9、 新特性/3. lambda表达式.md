### lambda表达式语法格式
```
[捕获列表](参数列表) -> 返回类型 { 函数体 }
```
1. 捕获列表：定义如何访问外部变量
2. 参数列表：和普通函数一致
3. 返回类型：可由编译器自动推导，复杂需自己设定

### 捕获列表
1. `[x]`：复制外部变量x到函数体
2. `[&x]`：直接引用
3. `[=] or [&]`：隐式表示，适用全局
4. `[=, &x]`：隐式表示，但是x特殊
5. `[*this]`：按值捕获，避免悬垂指针


定义一个一次性函数，在用到函数指针的地方就会可以用lambda
`[a或者是&a](int value函数的参数){函数的具体内容std::cout<<"value:"<<a<<std::endl;}`
1. a表示是值传递
2. &a表示的是引用传递
