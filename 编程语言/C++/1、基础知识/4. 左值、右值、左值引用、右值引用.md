### 左值
1. 只能出现在**赋值符号**的左边
2. 可以指示内存位置的值，可以被取地址
```
int x = 20; x是左值
int *p = &x;
```

### 左值引用

```
int x = 10;
int &ref = x;
int &ref = 10;//错误
```

### 右值
1. 只能出现在**赋值符号**的右边
2. 通常是指**临时变量**或**绑定不到内存的值**
3. 不能取地址
```
int x = 10; 10是右值
int y = x + 1; x+1是右值
int *p = &(x+1); 错误
```

### 右值引用
```
int &&ref = 20;正确
```
