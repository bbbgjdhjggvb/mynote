# 基本用法
```go
r := gin.Default()
// GET请求
r.GET("/index", func(c *gin.Context){

})

r. POST()
r.DELETE()
// PUT请求，更新信息
r.PUT()
```

# 客户端访问不存在页面
```go
r.NoRoute(func(c *gin.Context)){
	c.JSON(http.StatusNotFound, gin.H{
		
	})
}
```

# 当页面存在共同的前缀时可以使用路由组
```go
autoGroup := r.Goup("/auth")
{
	autoGroup.GET("/login", fun)
	autoGroup.POST("/register", fun)
}
```
