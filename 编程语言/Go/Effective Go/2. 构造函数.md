# 使用函数闭包解决参数过多的问题
## 作用
1. 调用者按需调用

## 实现步骤
1. 定义闭包函数控制可选参数
2. 在够造参数中调用返回的闭包函数
```go
type A struct{
	a int
	b int
	c int
	d int
	e int
}

type AOption func(*A)

func AWithc(c int)Aoption{
	return func(a *A){
		a.c = c
	}
}

func AWithd(d int)Aoption{
	return func(a *A){
		a.d = d
	}
}

// a、b参数是必填参数，其他参数是选填参数
func CreateA(a int, b int, options... AOption){
	a = &A{
		a: a,
		b: b,
	}

	for i in range options{
		options[i](a)
	}

	return a;
}
```
