# equals方法
1. 自反性
2. 对称性
3. 传递性
4. 一致性
```java
import java.util.Objects;

public class MyClass {
    // 示例字段
    private final String name;
    private final int id;
    private final double value;
    private final Object reference;

    public MyClass(String name, int id, double value, Object reference) {
        this.name = name;
        this.id = id;
        this.value = value;
        this.reference = reference;
    }

    // equals() 方法实现模板
    @Override
    public boolean equals(Object o) {
        // 1. 检查是否同一个对象
        if (this == o) return true;
        
        // 2. 检查类型是否匹配
        if (o == null || getClass() != o.getClass()) return false;
        
        // 3. 类型转换
        MyClass myClass = (MyClass) o;
        
        // 4. 比较所有关键字段（基本类型用 ==，对象用 Objects.equals()）
        return id == myClass.id &&
               Double.compare(myClass.value, value) == 0 &&
               Objects.equals(name, myClass.name) &&
               Objects.equals(reference, myClass.reference);
    }
}
```
# 重写hashcode函数
```java
    // hashCode() 方法实现模板
    @Override
    public int hashCode() {
        // 使用 Objects.hash() 包含所有关键字段
        return Objects.hash(name, id, value, reference);
    }
```

# 手动计算hashcode
```java
@Override
public int hashCode() {
    int result = 17; // 任意非零质数
    result = 31 * result + (name != null ? name.hashCode() : 0);
    result = 31 * result + id;
    long temp = Double.doubleToLongBits(value);
    result = 31 * result + (int) (temp ^ (temp >>> 32));
    result = 31 * result + (reference != null ? reference.hashCode() : 0);
    return result;
}
```
